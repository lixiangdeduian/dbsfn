# 前端UI设计说明

## 🎨 设计理念

基于现代医疗信息系统的设计趋势，采用清新、专业、易用的设计风格。

### 核心设计原则

1. **专业性** - 体现医疗行业的严谨与专业
2. **清晰性** - 信息层次分明，易于阅读
3. **舒适性** - 柔和的配色，降低视觉疲劳
4. **效率性** - 简化操作流程，提升工作效率

## 🎨 配色方案

### 主色调
- **主蓝色**: `#1e88e5` - 专业、信任
- **辅助蓝**: `#1976d2` - 深蓝，用于强调
- **浅蓝背景**: `#f0f9ff` - 柔和背景

### 辅助色
- **成功绿**: `#4caf50` - 积极、健康
- **警告黄**: `#f59e0b` - 提醒
- **错误红**: `#ef4444` - 警告
- **中性灰**: `#64748b` - 辅助文字

### 背景色
- **主背景**: 线性渐变 `#f5f7fa` → `#e8f4f8`
- **卡片背景**: 白色 `#ffffff`
- **区块背景**: 
  - 蓝色区块: `#f0f9ff` → `#e0f2fe`
  - 黄色区块: `#fef3c7` → `#fde68a`
  - 绿色区块: `#f0fdf4` → `#dcfce7`

## 📐 布局优化

### 侧边栏
- **宽度**: 240px（原200px）
- **样式**: 白色背景，右侧阴影
- **菜单项**: 增大间距，添加图标

### 顶部导航
- **高度**: 64px
- **背景**: 蓝色渐变
- **阴影**: 下方2px阴影，增强层次感

### 内容区域
- **外边距**: 24px
- **内边距**: 32px
- **圆角**: 12px
- **阴影**: 柔和阴影效果

## 🎯 组件优化

### 卡片（Card）
```css
- 圆角: 12px
- 阴影: 0 2px 8px rgba(0, 0, 0, 0.06)
- 悬浮效果: 向上移动4px + 阴影加深
- 边框: 1px solid #e8f4f8
```

### 表格（Table）
```css
- 表头背景: 浅蓝色渐变
- 表头文字: 深蓝色加粗
- 偶数行背景: #fafbfc
- 悬浮行背景: #f0f9ff
- 圆角: 8px（整体表格）
```

### 按钮（Button）
```css
主按钮:
- 背景: 蓝色渐变
- 阴影: 0 2px 8px rgba(30, 136, 229, 0.3)
- 悬浮: 向上移动1px + 阴影加深
- 圆角: 8px
```

### 输入框（Input）
```css
- 圆角: 8px
- 边框: 1.5px solid #e0e7ff
- 聚焦: 蓝色边框 + 蓝色阴影
```

### 统计卡片（Statistic Card）
```css
- 图标: 大号emoji，渐变色
- 数值: 36-40px，加粗，渐变色
- 标题: 15-16px，中等粗细
- 悬浮: 卡片上浮 + 阴影
```

## 🌈 特殊效果

### 渐变效果
1. **数值渐变**: 用于统计数字，增强视觉冲击
2. **背景渐变**: 柔和的渐变背景，避免单调
3. **按钮渐变**: 主按钮使用渐变增强层次

### 阴影效果
1. **卡片阴影**: 轻微阴影，增强悬浮感
2. **按钮阴影**: 强调可点击性
3. **悬浮阴影**: 交互反馈

### 动画效果
1. **悬浮动画**: transform translateY(-4px)
2. **过渡动画**: transition all 0.3s ease
3. **点击反馈**: 轻微缩放

## 📝 表单设计

### 表单分组策略
1. **基本信息** - 蓝灰色背景
2. **联系方式** - 浅蓝色背景
3. **紧急联系人** - 浅黄色背景
4. **医疗信息** - 浅红色背景

### 表单布局
- 使用栅格系统（Row/Col）
- 相关字段横向排列
- 大字段独占一行
- 适当分组和留白
